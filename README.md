# CCNA (200-301) Networking Practice Repository
# Overview

This repository documents my hands-on learning journey for the CCNA (200-301) certification. It focuses on building strong networking fundamentals through practical labs, configurations, and troubleshooting exercises.

All practice is based on the Jeremy’s IT Lab – CCNA 200-301 Complete Course, a widely recognized resource for learning industry-relevant networking concepts.

The purpose of this repository is to demonstrate practical skills, consistency, and problem-solving ability, making it suitable for internship and fresher networking roles.
